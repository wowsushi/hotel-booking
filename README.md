# Hotel booking
線上旅館訂房系統，目前只做出與訂房流程有關的功能，

## 使用技術
- React
- React Router
- SCSS
- Axios
- rc-calender 日曆插件
- HTML 5
- JavaScript

## 功能
- 可以瀏覽全部房型
- 可以瀏覽特定房行細節
- 可以顯示訂房價格總價
- 可以預訂旅館( 限90天內 )
- 可以顯示訂房成功或失敗
